# MotionForge Roadmap

MotionForge — Framer Motion + Forge konseptli ultra hafif, performans odaklı animasyon kütüphanesi.

## Vision
Zero-runtime, build-time codegen mimarisi ile en küçük bundle size ve gerçek 60fps+ animasyonlar üretmek.

TypeScript-first, tree-shakable ve Tailwind ile doğal entegre çalışan modern bir motion engine.

## v0.1 (MVP)
- Core API: `forge()` minimal taslak
- Zero-runtime codegen proof-of-concept
- Basic spring physics engine
- `examples/react-basic` demo
- README + ROADMAP + LICENSE
